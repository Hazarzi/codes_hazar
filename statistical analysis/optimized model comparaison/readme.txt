Two scripts for algorithm performance comparison. Scores are obtained by 5x2 fold CV for seven machine learning/ANN algorithms: Multiple Linear Regression, k-Nearest Neighbors, Support Vector Regression, Random Forests Regression, Extreme Gradient Boosting, Multilayer Perceptron and an TensorFlow ANN.

repeated-anova.py : Repeated-anova for identifying significiant differences between multiple model.

tukey-hsd.py : Tukey's HSD test for pairwise comparaison with corrected p-values.
